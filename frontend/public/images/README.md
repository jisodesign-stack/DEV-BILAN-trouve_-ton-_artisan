# Images du projet

Placez les fichiers d'images suivants dans ce dossier :

## Logo
- **logo.png** : Logo principal du site "Trouve ton artisan !"
  - Utilisé dans le header
  - Format recommandé : PNG avec transparence
  - Hauteur recommandée : 100-150px

## Favicon
- **favicon.ico** : Favicon du site (à placer dans `/public/`)
  - Icône de l'artisan fournie dans le brief
  - Tailles : 16x16, 32x32, 64x64

- **favicon-192.png** : Icône pour PWA et Apple Touch
  - Taille : 192x192 pixels

- **favicon-512.png** : Grande icône pour PWA
  - Taille : 512x512 pixels

## Image par défaut artisan
- **default-artisan.jpg** : Image placeholder pour les artisans sans photo
  - Format : JPG ou PNG
  - Dimensions recommandées : 400x400 pixels

## Instructions

1. Convertir le logo fourni en PNG et le nommer `logo.png`
2. Convertir le favicon (petit artisan) aux différentes tailles
3. Placer les fichiers dans ce dossier

Le favicon.ico doit être placé à la racine du dossier `/public/`
